# RandomSage 🌟

RandomSage is a React-based web application that generates random advice quotes using the Advice Slip API. Users can get a new piece of advice instantly with a single click, making the experience simple, engaging, and interactive.

## 🚀 Features

- Fetches random advice from an external API
- Generate new advice with a single click
- Responsive and clean user interface
- Real-time API integration
- Fast performance powered by Vite

## 🛠️ Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS
- REST API

## 🌐 API Used

- Advice Slip API


## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Vishal-git123/RandomSage.git
```

Navigate to the project folder:

```bash
cd RandomSage
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## 🎯 How It Works

1. The application sends a request to the Advice Slip API.
2. A random advice quote is returned.
3. React updates the UI dynamically.
4. Clicking the button fetches a new advice quote instantly.
